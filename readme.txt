== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wp-responsive-preview` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to a post or page and click the "Responsive Preview" button
